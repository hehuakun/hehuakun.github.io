在小端序的机器中，如果 ~~~~ union X { int x; char y[4]; }; ~~~~ 如果 ~~~~ X a; a.x=0x11223344;//16进制 ~~~~ 则：______ a.y[0]=11 a.y[1]=11 a.y[2]=11 a.y[3]=11 a.y[0]=22 a.y[3]=22 小端序，即低字节在低地址，高字节在高地址
